# 项目总览

## 项目名称

企业知识库智能体系统

## 项目定位

这是一个面向求职展示和后续企业化扩展的高完成度 AI 应用项目。它以企业知识库问答为主线，在 RAG 基础上逐步演化出任务路由、工具选择、LangGraph 工作流、运行记录、反馈闭环和 HITL。

项目目标不是做一个单点功能 Demo，而是展示：

- 智能体主流程设计能力
- 检索增强应用构建能力
- 服务化与工程化落地能力
- 可观测性与可扩展性设计能力

同时需要明确边界：

- 当前版本适合定义为 Agent 系统原型或作品集项目
- 当前版本不应包装为已经生产落地的企业级平台
- 企业化能力属于后续扩展方向，而不是当前版本已完成事实

## 当前能力总览

### 知识库能力

- 知识库创建与管理
- 文档上传与处理
- `txt / md / pdf / docx` 支持
- 多策略分块
- 向量化与向量检索

### 问答能力

- 基础 RAG 问答
- 自动选库
- 跨库检索
- 流式输出
- 会话与消息持久化

### Agent 能力

- 任务路由：`qa / summary / comparison`
- 查询改写
- 改写评估
- 工具选择
- 补充检索
- 检索后重排
- LangGraph 工作流

### 可观测性与质量能力

- `agent_runs` 运行记录
- 工作流轨迹查看
- 用户反馈
- 反馈统计
- 负反馈样本分析

### 人工协作能力

- 高风险人工审核建议
- `pending_review`
- 审核通过 / 驳回

## 技术栈

- 后端：FastAPI、SQLAlchemy、PostgreSQL、pgvector、Redis
- 智能体与检索：LangGraph、LangChain、自定义 Agent Tools
- 前端：Vue 3、Vite
- 部署：Docker、Docker Compose

## 当前项目结论

当前项目已经完成了一个完整 Agent 项目的主体骨架，足够用于：

- 求职项目展示
- 项目讲解
- 简历表达
- 后续企业级能力扩展

它目前最适合被定义为：

`高完成度 AI 应用项目 + Agent 系统原型 + 企业级扩展底座`

不建议对外表述为：

- 企业生产系统
- 已完成企业级落地
- 可直接用于正式业务环境
