# 开发日志

本文档记录项目开发过程中的关键阶段、重要改动和每一批工作的工程意义。它不是逐行变更日志，而是阶段性推进记录。

## 阶段 1：项目骨架

- 建立 `FastAPI + Vue + Docker Compose + PostgreSQL + Redis` 基础骨架
- 初始化前后端目录、环境变量、健康检查和基础页面
- 建立 `docs/` 文档体系

## 阶段 2：知识库模块

- 新增知识库、文档、分块相关模型
- 完成知识库创建、文档上传、文档处理、文档列表与分块查看
- 支持 `txt / md / pdf / docx`
- 引入多策略分块和向量化

## 阶段 3：RAG 问答与会话

- 完成检索、问答、流式问答
- 新增会话和消息持久化
- 前端完成知识库页和对话页基础联调

## 阶段 4：任务路由与自动选库

- 引入 `qa / summary / comparison` 路由
- 引入自动知识库选择和跨库检索
- 将结果展示到前端

## 阶段 5：Agent Tools 与 LangGraph

- 抽出 Agent Tools
- 将执行过程切换为 LangGraph 工作流
- 引入查询改写、改写评估、工具选择、检索质量判断、补充检索和重排

## 阶段 6：运行记录与可观测性

- 新增 `agent_runs`
- 增加运行记录查询接口和前端运行记录页
- 对话可直接跳到本次运行记录

## 阶段 7：反馈闭环与 Eval 基础

- 新增用户反馈
- 新增反馈统计
- 新增负反馈样本回放
- 新增多维聚合

## 阶段 8：HITL 建议、暂停与恢复

- 新增高风险审核建议
- 新增 `pending_review`
- 支持审核通过与驳回
- 审核结果可回写到对话和运行记录
