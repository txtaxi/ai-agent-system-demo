# 阶段 1：项目骨架

## 目标

完成企业知识库智能体系统的第一版工程骨架，为后续知识库、RAG 和 Agent 能力提供可扩展项目结构。

## 范围

本阶段主要完成：

- 后端基础目录
- FastAPI 入口
- 健康检查接口
- 前端 Vue 3 + Vite 基础目录
- Docker Compose 基础编排
- 项目计划书和文档体系

本阶段不实现：

- 文档解析
- 分块
- 检索
- 问答
- Agent 工作流

## 实际产出

### 后端

- `backend/app/main.py`
- `backend/app/api/router.py`
- `GET /api/v1/health`

### 前端

- Vue 3 + Vite 基础结构
- 首页、知识库页、对话页初始路由

### 部署

- `docker-compose.yml`
- `.env.example`

### 文档

- 项目计划书
- `docs/` 目录

## 阶段结论

阶段 1 的核心价值不是业务能力，而是：

- 项目能启动
- 目录能扩展
- 文档能跟进

它为后续阶段提供了一个稳定的工程底座。
